# 材质添加说明

## 材质文件位置

材质文件应该放在以下目录：
```
src/main/resources/assets/d-core/textures/
├── item/          # 物品材质
│   ├── core_fragment.png
│   ├── core_crystal.png
│   └── enhanced_core.png
├── block/         # 方块材质（如果有方块）
└── entity/        # 实体材质（如果有实体）
```

## 像素要求

### 标准要求
- **物品材质**：通常是 **16x16 像素**（推荐）
- **方块材质**：通常是 **16x16 像素**
- **实体材质**：根据实体大小，通常是 16x16、32x32、64x64 等

### 像素规则
1. **必须是 2 的幂次方**：16、32、64、128、256 等
2. **推荐尺寸**：
   - 物品：16x16（最常用）
   - 高分辨率物品：32x32 或 64x64（需要资源包支持）
   - 方块：16x16
   - 实体：根据模型大小，通常是 32x32 或更大

### 注意事项
- 如果使用高分辨率材质（如 32x32），需要玩家安装支持高分辨率的资源包
- 默认情况下，Minecraft 使用 16x16 的材质
- 材质文件必须是 **PNG 格式**

## 材质文件命名

材质文件名必须与模型 JSON 文件中引用的路径匹配：

**模型文件** (`models/item/core_crystal.json`):
```json
{
  "parent": "item/generated",
  "textures": {
    "layer0": "d-core:item/core_crystal"
  }
}
```

**对应的材质文件**：
- 路径：`textures/item/core_crystal.png`
- 完整引用：`d-core:item/core_crystal` → `assets/d-core/textures/item/core_crystal.png`

## 如何创建材质

### 方法 1：使用图像编辑软件
1. 打开 Photoshop、GIMP、Aseprite 等图像编辑软件
2. 创建新图像，尺寸设为 16x16 像素（或 32x32、64x64）
3. 绘制你的材质
4. 保存为 PNG 格式
5. 放到对应的 `textures/` 目录下

### 方法 2：使用在线工具
- **Minecraft Texture Editor**：在线材质编辑器
- **Blockbench**：3D 模型和材质编辑器

### 方法 3：使用占位材质
如果暂时没有材质，可以使用：
- 纯色方块（如红色、蓝色）
- 简单的几何图形
- 从其他模组或原版材质中临时借用（仅用于测试）

## 当前项目需要的材质文件

根据 `ModItems.java` 中定义的物品，需要以下材质：

1. `textures/item/core_fragment.png` - 核心碎片
2. `textures/item/core_crystal.png` - 核心水晶  
3. `textures/item/enhanced_core.png` - 强化核心

## 材质示例（占位符）

如果暂时没有材质，可以创建简单的占位材质：
- 16x16 像素
- 纯色背景 + 简单图案
- PNG 格式，透明背景（如果需要）

## 多图层材质

如果模型使用多个图层（如 `layer0`, `layer1`），需要为每个图层提供对应的材质文件。
