材质文件说明
============

此目录用于存放物品材质文件（PNG 格式）

需要的材质文件：
1. core_fragment.png - 核心碎片材质
2. core_crystal.png - 核心水晶材质
3. enhanced_core.png - 强化核心材质

材质要求：
- 尺寸：16x16 像素（推荐）或 32x32、64x64 等（必须是 2 的幂次方）
- 格式：PNG
- 命名：必须与模型 JSON 文件中引用的名称一致

如何创建：
1. 使用图像编辑软件（如 Photoshop、GIMP、Aseprite）创建 16x16 像素的图像
2. 绘制你的材质
3. 保存为 PNG 格式
4. 放到此目录下

如果没有材质，游戏会显示紫色黑色方块（缺失材质提示）
